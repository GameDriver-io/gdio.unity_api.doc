# ApiClient.Scroll method

Use this function to simulate mouse wheel scrolling. NOTE: The new input system values are a factor of 120

```csharp
public bool Scroll(float x, float y, ulong frame_count, int timeout = 30)
```

| parameter | description |
| --- | --- |
| x | The x value of the scroll vector for the mouse, usually 0. |
| y | The y value of the scroll vector for the mouse, typically -1 (-120) &lt;= y &lt;= +1 (120) |
| frame_count | The numer of frames to activate scroll event. |
| timeout | The time to wait for the agent to process the Scroll request. |

## Return Value

TRUE if the GameDriver agent was able to successfully process the request.
