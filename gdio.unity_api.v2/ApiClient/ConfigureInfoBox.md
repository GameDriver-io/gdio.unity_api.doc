# ApiClient.ConfigureInfoBox method

Updates the configuration of the infobox. Can be used to hide/unhide it and change position and scale.

```csharp
public virtual void ConfigureInfoBox(bool enabled = true, bool toggleInfo = false, 
    int canvasRenderMode = 0, Vector3 position = null, Vector2 scale = null, int timeout = 30)
```

| parameter | description |
| --- | --- |
| enabled | Whether the infobox should be shown or not. |
| toggleInfo | True to togle the dropdown of the infobox, false to leave it alone. This is a toggle, not a true/false for show not show. |
| canvasRenderMode | The render mode of the canvas. 0 = Screen Space Overlay, 1 = Screen Space Camera, 2 = World Space. |
| position | The rect position of the infobox. |
| scale | Rhe scale of the infobox. |
| timeout | The number of seconds to wait for a response that the ConfigureInfoBox request was processed. |
