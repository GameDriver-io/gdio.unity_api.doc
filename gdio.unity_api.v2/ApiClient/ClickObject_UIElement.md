# ApiClient.ClickObject_UIElement method

Use this function to perform in-game mouse-clicks with the UIToolkit.

```csharp
public bool ClickObject_UIElement(MouseButtons buttonId, string hierarchyPath, 
    string cameraHierarchyPath, ulong clickFrameCount, KeyCode[] modifiers, int timeout = 30)
```

| parameter | description |
| --- | --- |
| buttonId | The mouse button to use for the click operation. See MouseButtons. |
| hierarchyPath | The HierarchyPath of the element to click. |
| cameraHierarchyPath | The HierarchyPath to the Camera rendering the object. An empty string uses Camera.main |
| clickFrameCount | The number of frames to click the specific position. |
| modifiers | The modifiers to be applied when clicking. |
| timeout | The number of seconds to wait for a response that the Click request was processed. Input requests process asynchronously, so a response does NOT imply the input operation has completed. |

## Examples

```csharp
api.ClickUIToolkit(MouseButtons.LEFT, 0, 0, 30, new Keycode[] { Keycode.LeftShift } 30); //Clicks the left mouse button at 0,0 for 30 frames with Shift modifier pressed
```
