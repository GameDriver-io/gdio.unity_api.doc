# ApiClient.SetXRDeviceMode method

Used to change between controller and hand modes in XR if one of the used modes is not working or being rendered. Mostly for OVR. Set by default to both: XRDeviceModes.CONTROLLER &#x7C; XRDeviceModes.HAND. Ex: SetXRDeviceModes("GDIOLeftHand", XRDeviceModes.HAND) or SetXRDeviceModes("GDIOLeftHand", XRDeviceModes.CONTROLLER &#x7C; XRDeviceModes.HAND).

```csharp
public bool SetXRDeviceMode(string deviceName, XRDeviceModes deviceMode)
```

| parameter | description |
| --- | --- |
| deviceName | The name of the device to switch mode. |
| deviceMode | The mode of the device to switch into. |

## Return Value

True if the device is found and the mode is switched, an exception and false otherwise.
