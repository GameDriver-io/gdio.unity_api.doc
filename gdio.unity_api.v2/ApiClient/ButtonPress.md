# ApiClient.ButtonPress method (1 of 2)

Use this function to send arbitrary button states to the game. Defaults to LEFT ALT/CTRL/SHIFT/WINDOWS(COMMAND)

```csharp
public bool ButtonPress(string buttonId, ulong numberOfFrames, int timeout = 30)
```

| parameter | description |
| --- | --- |
| buttonId | Button input to press. Name as defined in the Input Manager (Old) |
| numberOfFrames | The number of frames to press and hold the button for. |
| timeout | The timeout in seconds to wait for a response that the request was processed by the GameDriver agent. |

## Return Value

TRUE if the GameDriver agent successfully processed the request.

## Examples

```csharp
//Press the Fire1 then Fire2 buttons in sequence
            api.ButtonPress("Fire1", 100);
            api.Wait(300);
            api.ButtonPress("Fire2", 100);
            api.Wait(300);
            
            //A frame count of 0 can be used to uniquely and seperately trigger ButtonDown and ButtonUp events.
            api.ButtonPress("Jump", 0);
            api.Wait(500);
            api.ButtonPress("Jump", 0);
```
