# ApiClient.Vector3InputEvent method

Use this function to send arbitrary Vector3 Event states to the game.

```csharp
public bool Vector3InputEvent(string inputControlPath, Vector3 value, ulong numberOfFrames, 
    int timeout = 30)
```

| parameter | description |
| --- | --- |
| inputControlPath | The Input Control Path for the New Input System |
| value | Value of the Vector3 event |
| numberOfFrames | The number of frames to press and hold the axis for. |
| timeout | The timeout in seconds to wait for a response that the request was processed by the GameDriver agent. |

## Return Value

TRUE if the GameDriver agent successfully processed the request.

## Examples

```csharp
api.Vector3InputEvent("OculusLeftHand/devicePosition", new Vector3(0, 0, 5), 2); //Move the left controller to the coordinates relative to the HMD.
```
