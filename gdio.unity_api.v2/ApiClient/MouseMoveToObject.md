# ApiClient.MouseMoveToObject method

Move the mouse to the center of a specific object, identified by the HierarchyPath.

```csharp
public bool MouseMoveToObject(string objectHierarchyPath, ulong frameCount, 
    bool waitForObject = true, bool waitForEmptyInput = true, int timeout = 60)
```

| parameter | description |
| --- | --- |
| objectHierarchyPath | The HierarchyPath to move the mouse pointer to. |
| frameCount | The number of frames to compelte the mouse move over. |
| waitForObject | If TRUE, wait for the object to exist so that mouse move can complete successfully. |
| waitForEmptyInput | Wait for the empty input event to be returned from the GameDriver agent, before returning from the method. |
| timeout | The timeout in seconds to wait for a response that the request was processed by the GameDriver agent. If waitForEmptyInput is TRUE, then the method call will wait the timeout alotment to recieve the event. |

## Return Value

TRUE if the method call was successfully processed by the GameDriver agent.

## Examples

```csharp
//Moves the mouse to the center of the object named "Cylinder" over 300 frames
            api.MouseMoveToObject("//*[@name='Cylinder']", 300, true, true);
```
