# ApiClient.CaptureScreenshot method

Use this function to capture a screenshot of the Game under test.

```csharp
public string CaptureScreenshot(string filename, bool storeInGameFolder = false, 
    bool overwriteExisting = false, int timeout = 60)
```

| parameter | description |
| --- | --- |
| filename | The filename, and path, of the screen capture. |
| storeInGameFolder | If true, this won't transfer the screen capture back to the client and instead will save it on the device where the game is running. |
| overwriteExisting | If true, the operation will overwrite the file if it already exists. |
| timeout | The number of seconds to wait for a response that the CaptureScreenshot request was processed. |

## Examples

```csharp
api.CaptureScreenshot(@"c:\temp\screen1.png", false, true);
```
